# Tree Packing Challenge

## Overview
Repository for the Tree Packing Kaggle Challenge.

## Structure
- `data/`: Contains raw and processed data (ignored by git).
- `src/`: Source code for data processing, feature engineering, and modeling.
- `scripts/`: Executable scripts for running pipelines.
- `config/`: Configuration files and path definitions.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Place raw data in `data/raw/`.
